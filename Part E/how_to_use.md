# How to Use

## Open `app.py`
Open the `app.py` file in your code editor.

## Update the connection settings
In `app.py`, enter the correct values for:
- Username
- Password
- DSN

## Make sure the database is ready
Before running the app, the database must already be:
1. running
2. loaded with the project tables
3. loaded with the sample data

## Run the application
Open **Terminal**, go to the project folder, and run:

```bash
python3 app.py
